# Vendored from https://github.com/TheBoredTeam/boring.notch (GPL-3.0)

Aura's Dynamic Island notch UI is the Boring Notch `ContentView` stack, integrated via `BoringNotchHost`.

Aura-specific features kept outside the notch:
- Menu bar Command Center (Pomodoro, Deep Work, Stopwatch, Todo)
- Floating widgets / PanelManager
- Aura SettingsView

Not ported (shimmed or excluded): Sparkle updater, SkyLight lock-screen window, Lottie custom visualizers, XPC brightness helper, YouTube Music controller, onboarding/what's-new.
