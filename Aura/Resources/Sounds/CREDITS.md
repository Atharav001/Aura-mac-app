# Timer alert sounds

Bundled alert tones under `Aura/Resources/Sounds/` are short system alert samples derived from macOS `/System/Library/Sounds` (Glass, Ping, Tink, Bottle, Purr), renamed for Aura:

| File | Display name |
|------|----------------|
| `bell-glass.aiff` | Glass Bell |
| `bell-ping.aiff` | Ping |
| `bell-tink.aiff` | Tink |
| `bell-bottle.aiff` | Bottle |
| `bell-soft.aiff` | Soft Chime |

Users may also add a custom MP3/WAV/AIFF via Settings → Timers → Timer alert sound.
